# dotfiles

## Quick Start

With [chezmoi](https://github.com/twpayne/chezmoi) installed:

```
chezmoi init -a TimoSutterer
```

On a new machine:

```
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply TimoSutterer
```
